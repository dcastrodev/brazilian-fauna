# Brazilian Fauna Institute 🐆
<!-- 🐾 -->

O Fauna Brazilian Institute é um projeto autoral, desenvolvido do zero — desde a concepção do layout até a implementação do código ( sem uso de frameworks). O projeto apresenta um instituto dedicado à proteção de espécies ameaçadas, focado na conscientização e na captação de recursos para centros de conservação na fauna brasileira.

<a href="https://dcastrodev.github.io/brazilian-fauna/">🔗 Live Demo</a>

## Objetivo do Projeto

O foco principal foi aplicar de forma prática e harmônica os conceitos fundamentais de Front-end, transformando teoria em uma interface funcional e visualmente atraente, incluindo:

- Botões interativos
- Uso de videos como background (header)
- Posicionamento de elementos utilizando o Flexbox
- Responsividade (Pc & mobile).
- Design de Interface (UI): Estilização personalizada de fontes, paleta de cores temática e espaçamentos equilibrados.

## Desafios

O maior desafio deste projeto foi a etapa de **Design e Curadoria**. Diferente de seguir um tutorial, criar um layout próprio exige decisões sobre:
- Escolha de mídias (imagens e vídeos)
- Equilíbrio visual que garanta legibilidade das informações e uma experiência de usuário fluída e intuítiva.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Funcionalidades

- Informações sobre o Brazilian Fauna Institute e suas iniciativas.
- Estilização personalizada para cada espécie diferente, com informações.
- Animações interativas nos botões.
- Design estilizado usando Flexbox.
